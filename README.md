# Decoded

**What they said. What it actually means.**

Decoded is a concept landing page for a tool that translates corporate earnings-call jargon into plain English. Built for the IronLabs AI Hackathon (GTM & Creative Execution track).

## The Idea

Companies often soften bad news with vague corporate-speak — "navigating near-term headwinds" instead of "sales are down with no fix in sight." Decoded would let anyone paste an earnings-call transcript and get:

- Flagged "hedge" phrases, struck through
- A plain-English translation for each
- An overall "Confidence Index" score for the call

## What's in this repo

- Landing page (HTML/React) — hero, jargon→translation document card, features, stats, signup form
- Built with [Emergent](https://plain-speak-28.emergent.host/)

## Design
